# 📚 منصة الرياضيات الذكية

## 🎯 نبذة عن المنصة
منصة تعليمية تفاعلية لتعليم الرياضيات بطريقة ممتعة وفعّالة، تتضمن:
- ✅ فيديوهات تعليمية واضحة
- ✅ أسئلة تفاعلية مع شرح مفصل
- ✅ تقييم فوري للإجابات
- ✅ نظام تحديات ممتع
- ✅ واجهة جميلة وسهلة الاستخدام

## 📋 محتوى المنصة

### المرحلة الابتدائية
- **الصف الأول**: 6 دروس في الترم الأول + دروس في الترم الثاني
- **الصف الثاني**: دروس في الجمع والطرح والضرب والكسور
- **الصف الثالث**: دروس في العمليات الحسابية والأشكال الهندسية
- **الصف الرابع**: دروس متقدمة في الضرب والقسمة والكسور العشرية
- **الصف الخامس**: النسبة والتناسب والأعداد العشرية
- **الصف السادس**: الجبر والإحصاء والأشكال الهندسية

### المرحلة الإعدادية
- **الصف الأول الإعدادي**: المجموعات والمعادلات والهندسة
- **الصف الثاني الإعدادي**: التحليل والخطوط المتوازية والتشابه
- **الصف الثالث الإعدادي**: الأعداد الحقيقية والنسب المثلثية والاحتمالات

## 🚀 كيفية الاستخدام المحلي

### الخيار 1: فتح الملف مباشرة
```
1. افتح ملف منصة_محسنة.html بمتصفحك
2. اختر المرحلة (ابتدائي/إعدادي)
3. اختر الصف والترم والدرس
4. شاهد الفيديو وأجب على الأسئلة
```

### الخيار 2: استخدام خادم محلي (موصى به)
```bash
# استخدام Python 3
python -m http.server 8000

# ثم افتح المتصفح على:
http://localhost:8000/منصة_محسنة.html
```

## 🌐 نشر المنصة أونلاين

### الطريقة 1: GitHub Pages (مجاني)

1. **انشئ حساب GitHub** (إذا لم يكن لديك)
   - اذهب إلى github.com

2. **انشئ مستودع جديد**
   - اسم المستودع: `math-platform` أو أي اسم تريده
   - اجعله عام (Public)

3. **رفع الملفات**
   ```bash
   # استخدم Git أو رفع مباشر عبر الواجهة
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/math-platform.git
   git push -u origin main
   ```

4. **فعّل GitHub Pages**
   - اذهب إلى Settings > Pages
   - اختر: Deploy from a branch
   - البراينش: main
   - المجلد: root
   - اضغط Save

5. **الرابط النهائي**
   ```
   https://YOUR_USERNAME.github.io/math-platform/منصة_محسنة.html
   ```

### الطريقة 2: Netlify (مجاني وسهل)

1. **انشئ حساب على Netlify**
   - اذهب إلى netlify.com
   - سجّل دخول بـ GitHub

2. **رفع الملفات**
   - اضغط "New site from Git"
   - اختر GitHub وحدد المستودع
   - اضغط Deploy

3. **الرابط سيكون مثل**
   ```
   https://your-domain.netlify.app/منصة_محسنة.html
   ```

### الطريقة 3: Vercel (مجاني وسريع)

1. **انشئ حساب على Vercel**
   - اذهب إلى vercel.com

2. **رفع المشروع**
   - اضغط "Import Project"
   - اختر GitHub

3. **الرابط**
   ```
   https://your-project.vercel.app/منصة_محسنة.html
   ```

## 🔧 التخصيص والتطوير

### إضافة دروس جديدة
```javascript
{
  title: 'اسم الدرس',
  video: 'رابط الفيديو أو المسار المحلي',
  questions: [
    {
      question: 'نص السؤال',
      options: ['خيار 1', 'خيار 2', 'خيار 3', 'خيار 4'],
      correct: 0, // رقم الإجابة الصحيحة
      explanation: {
        correct: '✅ رسالة التصحيح',
        wrong: '❌ رسالة الخطأ'
      }
    }
  ]
}
```

### تغيير الألوان
انظر إلى قسم `:root` في CSS لتخصيص الألوان

### تعطيل الفيديوهات المحلية
إذا كانت الفيديوهات المحلية لا تعمل، استخدم روابط YouTube بدلاً منها:
```javascript
video: 'https://www.youtube.com/embed/VIDEO_ID'
```
